INSTRUKSI:
==========

1. Letakkan file CSV Anda di folder ini (folder import/)
2. Pastikan nama file: "user - Sheet1 (1).csv"
   Atau sesuaikan nama file di script import_users.php

3. Format CSV harus:
   npk,name,unit_kerja,nomor_telepon,email,username,password_hash

4. Setelah file CSV ada di folder ini, jalankan:
   http://infoadkor.com/permintaandof/import_users.php

Catatan:
- File CSV akan dibaca dari folder ini
- Pastikan file CSV ada sebelum menjalankan script import
