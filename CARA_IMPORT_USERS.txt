CARA IMPORT USERS DARI CSV
==========================

LANGKAH 1: PINDAHKAN FILE CSV
------------------------------
1. Buka folder Downloads Anda
2. Cari file: "user - Sheet1 (1).csv"
3. Copy file tersebut
4. Paste ke folder: permintaandof/import/
   (Lokasi lengkap: F:\apps\laragon\www\permintaandof\import\)


LANGKAH 2: JALANKAN IMPORT
---------------------------
1. Buka browser
2. Akses: http://infoadkor.com/permintaandof/import_users.php
3. Tunggu proses import selesai
4. Lihat hasil import di halaman


CATATAN:
--------
- File CSV harus ada di folder import/ sebelum menjalankan script
- Nama file harus: "user - Sheet1 (1).csv"
- Jika nama file berbeda, edit di import_users.php baris 20


TROUBLESHOOTING:
----------------
Jika error "File CSV tidak ditemukan":
✓ Pastikan file sudah di folder import/
✓ Pastikan nama file benar
✓ Cek permission folder import/
